# CinemAi Neo Assets

This directory contains screenshots and visual assets for documentation.

## Landing Page Screenshots

To add the landing page screenshot:

1. Visit https://cinemai-bice.vercel.app
2. Take a full-page screenshot
3. Save as `landing-page.png` in this directory
4. Recommended dimensions: 1920x1080 or higher
5. Optimize for web (compress to < 500KB)

## Screenshot Guidelines

- Use high-quality captures (Retina/HiDPI if available)
- Include the full viewport showing key features
- Use PNG format for UI screenshots
- Compress images to keep repository size manageable
- Update screenshots when UI changes significantly

## Tools for Screenshots

- **macOS**: Cmd + Shift + 4 (selection) or Cmd + Shift + 3 (full screen)
- **Windows**: Windows + Shift + S
- **Linux**: Various tools (gnome-screenshot, scrot, etc.)
- **Browser Extensions**: Full Page Screen Capture, Awesome Screenshot
