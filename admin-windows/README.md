# CinemAi Admin - Windows 11 Application

Modern Windows 11 desktop application for CinemAi Neo administration.

## Quick Build

```bash
cd admin-windows && npm install && npm run dist
```

Output: `admin-windows/release/cinimaiadmin-1.0.0.exe`

## Features

- User Management & CRM
- Spam & Billing Control  
- Analytics & System Fees
- Contract & API Management
- Email System Integration
